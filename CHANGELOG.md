# Changelog

## 0.1.0 - 2026-06-25

Initial public version.

- Added the `clarity-map` Codex skill for ability, direction, capability-gap, and action confusion.
- Added a state machine to separate intake, mirror/clarify, diagnosis, experiment, follow-up, and close.
- Added a fixed delivery contract: one-sentence diagnosis, verified assets, candidate directions, capability gaps, minimum experiment, result card, and completion boundary.
- Added safety and scope boundaries for crisis, therapy-like use, legal/medical/compliance questions, and non-delegable decisions such as pricing or contract terms.
- Added closed-loop ending: final synthesis ends the round and only exposes `重新开始` as the built-in restart path.
- Added Canonical Diagnosis Card v1 with deterministic SVG rendering to prevent visual style drift.
- Added repository README, license, dependency declaration, and regression examples for ongoing maintenance.
